# 🧠 News4u – AI-Native News Experience

> “I can’t go back to reading news the old way.”

News4u is a Chrome Extension that transforms traditional news articles into an **interactive AI-powered experience** using Gemini.

---

## 🚨 Problem

Business news is still:
- Static text  
- Same for every user  
- Time-consuming  
- Hard to quickly understand  

👉 News is still delivered like **2005, not 2026**

---

## 💡 Solution

News4u converts any article into:

> 📰 Article → 🧠 Summary + 🎬 Video + 📊 Graph + 💬 Chat  

A faster, smarter, and more engaging way to consume news.

---

## ✨ Features

### 🧠 AI Summary
- Converts long articles into short, clear summaries  

---

### 🎬 AI Video Generator
- Turns news into animated videos  
- Uses images, subtitles, and AI narration  

#### Advanced Controls
- 🎭 Tone: Simple / Dramatic / Serious  
- 📱 Format: 16:9 + 9:16 (Reels ready)  
- ⚡ Speed: 1x / 1.5x / 2x  
- ⏯️ Pause / Resume / Mute  

---

### 📊 Data Visualization
- Extracts key numbers  
- Displays graphs for quick insights  

---

### 💬 Ask AI
- Chat with the article  
- Context-aware answers  

---

## 🛠️ Tech Stack

- **Frontend:** JavaScript, HTML, CSS  
- **Extension:** Chrome Extension APIs  
- **AI:** Gemini API  
- **Voice:** Web Speech API  
- **Images:** Unsplash  

---

## ⚙️ How It Works

1. Open a news article  
2. Extension injects AI buttons  
3. Click any feature  
4. Get instant AI output
